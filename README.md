## How to put it online (Netlify)
1. Go to **app.netlify.com** → **Add new site** → **Import an existing project** → **GitHub**.
2. Pick this repository.
3. Leave every box blank and click **Deploy**.
4. Done. Netlify gives you a live link.

To update the site later: upload your changed file to GitHub. Netlify republishes by itself.
